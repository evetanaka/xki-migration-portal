# XKI Migration Portal

Frontend pour la migration XKI (Ki Chain → Ethereum).

## Stack
- HTML5
- Vanilla JavaScript
- CSS3 (no framework)

## Features
- Connexion Keplr wallet
- Vérification d'éligibilité
- Signature de message Cosmos
- Soumission de claim
- Vérification de statut

## Deployment
Static files - deploy to any web server or CDN.

## Configuration
Update `API_BASE` in `js/app.js` to point to the backend API.
